Binary Search Tree

A binary search tree is a tree data structure, in which each node contains a key value, and the binary search tree invariant requires that the left subtree of a node contains only nodes with keys less than the node's key, and the right subtree of a node contains only nodes with keys greater than the node's key.

Implementation

Here are the following operations that can be performed on a binary search tree:

  • Insert: Insert a new key into the binary search tree.
  • Delete: Delete a key from the binary search tree.
  • Display(inorder): Display the nodes of the binary search tree in inorder traversal.
  • Search: Search for a key in the binary search tree.
  • BFS(Level wise print): Print the nodes of the binary search tree level-wise.
